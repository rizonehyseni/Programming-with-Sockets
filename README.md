# Programming-with-Sockets

Ky projekt implementon serverin dhe klientin në Node.js duke përdorur protokollin UDP, sipas kërkesave të lëndës Rrjetat Kompjuterike.
Qëllimi është të krijohet një sistem komunikimi ndërmjet disa pajisjeve në një rrjet real, ku serveri menaxhon kërkesat dhe klientët ndërveprojnë me të përmes komandave të ndryshme.
